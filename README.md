## Instalación

- Descargar el repositorio
  ```bash
  $ git clone https://github.com/carlossantesp/portafolio-elisa.git
  ```
- Instalar las dependecias

  **NPM**
  ```bash
  $ npm install
  ```

  **YARN**
  ```bash
  $ yarn install
  ```
- Ejecutar servidor de desarrollo

  **NPM**
  ```bash
  $ npm run dev
  ```

  **YARN**
  ```bash
  $ yarn dev
  ```
- Construir sitio web para el deploy

  **NPM**
  ```bash
  $ npm run export
  ```

  **YARN**
  ```bash
  $ yarn export
  ```
